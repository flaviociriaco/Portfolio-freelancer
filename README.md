# 🚀 Portfólio Freelancer 

Este projeto é um site de portfólio profissional criado para apresentar meus serviços como desenvolvedor freelancer.

## 📌 Sobre o Projeto

O objetivo deste site é divulgar meus serviços de criação de sites modernos, responsivos e profissionais para empresas, negócios locais e empreendedores.

O layout, estrutura visual e parte do código foram desenvolvidos com auxílio de Inteligência Artificial, sendo posteriormente ajustados, organizados e personalizados por mim para fins profissionais e aprendizado prático.
